# 1C bank application

В рамках работы с 1С:Конфигуратор было создано приложение, автоматизирующее бизнес-процесс брокерского обслуживания в банке. Были создана база данных информационной системы, печатные формы договоров брокерского обслуживания, согласия на обработку персональных данных, внутренние отчеты.
