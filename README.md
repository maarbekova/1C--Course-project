# 1C-Course-project

В рамках практической работы в 1С:Конфигуратор была создана конфигурация, автоматизирующая комплекс задач бизнес-процесса брокерского обслуживания в банке. Были создана база данных информационной системы, печатные формы договоров брокерского обслуживания, согласия на обработку персональных данных, внутренние отчеты.
