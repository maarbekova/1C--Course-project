# 1C bank application

В рамках практической работы в 1С:Конфигуратор была создано приложение, автоматизирующая бизнес-процесс брокерского обслуживания в банке. Были создана база данных информационной системы, печатные формы договоров брокерского обслуживания, согласия на обработку персональных данных, внутренние отчеты.
